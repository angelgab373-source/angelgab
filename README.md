# 📝 To-Do List en Python

Aplicación de consola para gestionar tareas de forma sencilla utilizando Python.

## 🚀 Funcionalidades

* Agregar nuevas tareas
* Ver lista de tareas
* Marcar tareas como completadas
* Eliminar tareas

## 🛠️ Tecnologías utilizadas

* Python
* Manejo de archivos con JSON

## ▶️ Cómo ejecutar el proyecto

1. Asegúrate de tener Python instalado
2. Descarga o clona este repositorio
3. Abre una terminal en la carpeta del proyecto
4. Ejecuta el siguiente comando:

```
python app.py
```

## 📂 Estructura del proyecto

* `app.py` → Programa principal
* `tareas.json` → Archivo donde se guardan las tareas

## 💡 Notas

Este proyecto fue desarrollado como práctica para aprender el uso de repositorios con Git y gestión básica de proyectos.
